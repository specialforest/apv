About APV

Overview

APV is native PDF Viewer for Android Platform.

If you ever encounter any problems with APV, please report bugs at http://code.google.com/p/apv/issues/entry. Those bug reports really help us improve APV.

APV is licensed under GPLv3.

APV project page is at apv.googlecode.com.
There's also a user group at groups.google.com/group/android-pdf-viewer.

Third party libraries and content

APV uses following third party software:

  • MuPDF library by Artifex Software, Inc. available at ccxvii.net/mupdf/
  • This software is based in part on the work of the Independent JPEG Group. JPEG library is available at www.ijg.org
  • FreeType2 library available at www.freetype.org

APV uses following third party media:

  • btn_zoom_up and btn_zoom_down drawables that are part of Android Platform
